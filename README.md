## âœ… Assignment Tasks Performed

### 1. Quantity Validation

* Restricted *Quantity* values to whole numbers between **1 and 10**
* Used **Circle Invalid Data** to identify incorrect entries

### 2. Total Price Highlighting

* Highlighted *Total Price* values greater than **$2000**
* Resolved `#VALUE!` errors using `IFERROR()`

### 3. Duplicate Order ID Check

* Checked for duplicate *Order IDs* using **Remove Duplicates**

### 4. Customer Name Split

* Split full names into **First Name** and **Last Name** using **Text to Columns** and **Flash Fill**

### 5. Conditional Row Highlighting

* Highlighted rows where:

  * Category = *Electronics*
  * Total Price > **$1000**

### 6. Region-Based Filtering

* Filtered dataset to display orders from the **North** region only

### 7. Quantity Error Correction

* Corrected text values (e.g., "Five") in the Quantity column using **Find & Replace**

### 8. Identifying Multiple Orders

* Highlighted customers with multiple orders using **Conditional Formatting**
* Classified customers as *Single Order* or *Multiple Orders* using a formula

**Result:** Customers who placed multiple orders were successfully highlighted.
